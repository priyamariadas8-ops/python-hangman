# Python Hangman Game

This is a simple Hangman game created using Python.

## About the Project

In this game, the computer selects a random word and the player has to guess the letters one by one.

The player has limited chances. If all letters are guessed correctly, the player wins.

## Features

- Random word selection
- Letter guessing
- Limited chances
- Win and lose conditions
- Beginner-friendly Python project

## Concepts Used

- Variables
- Lists
- Loops
- If-Else Conditions
- Random Module
- User Input

How to Run

Run the Python file using:

bash
python main.py
